Mis Proyectos

Aquí les comentare algunos proyectos de los que hago o hice parte.

Proyecto de Automatización

Se realizo la implementación de un sistema de monitoreo y control con herramientas de la I4.0 en una celda de manufactura flexible. Esta proyecto se realizo mediante la programación e interconectividad de mulples PLC, y el uso de node-red como herramienta de conectividad IoT para brindar información en tiempo real del proceso.

Drones

En la actualidad me encuentro realizando un estudio de operación de drones para uso principal en procesos de agricultura, con los drones se busca recolectar continuamente datos sobre los cultivos. Se realiza un procesamiento y estudio de la información obtenida y mediante algoritmos de aprendizaje automatico brindarle al cliente una información de utilidad.

Control de Temperatura

Se diseño un controlador de logica difusa para una sistema de temperatura por medio de Python y C. El controlador demostro estabilidad en relación a la variable, lo que proporciona repetibilidad y ahorro de energia.

Monitoreo en tiempo real

Mediante la opción de Web Server de un PLC marca Siemens, se realizo un dashboard para monitorear en tiempo real un simulación de una paletizadora industrial.
